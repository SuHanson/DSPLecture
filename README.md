# DSPLecture
-Think DSP in Python

This repo is for an introduction to Digital Signal Processing in Python.
Most of the code for this book is in Jupyter notebooks which are intended to run on [Google Colab](https://colab.research.google.com/). You may modify the downloaded code and run it on your computer if you have Jupyter environment on your computer. You can build this environment as the [official web site's](https://jupyter.org/) instruction.
